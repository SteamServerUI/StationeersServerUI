Server API Documentation

V2.X CURRENT

This server provides a complete REST API alongside the HTML interface. All UI actions map to API endpoints, allowing full programmatic control.

New in V2.X: Full Discord integration for server monitoring and management through your Discord server.

Available Endpoints

Server Control

  • GET
    /start
    Start the game server
  • GET
    /stop
    Stop the game server

Information

  • GET
    /output
    Retrieve server console output
  • GET
    /backups
    List available save files

Configuration

  • GET
    /config
    View server configuration
  • POST
    /saveconfig
    Save server configuration (Form data)

Save Management

  • GET
    /restore?index=123
    Restore from specified save index

Developer Challenge

Want to use the API programmatically? Reverse-engineer the form structure yourself.

Tip: Examine the network requests in your browser's developer tools when using the UI.
