About the Server API

This server provides a complete REST API alongside the HTML interface. All UI actions map to API endpoints, allowing full programmatic control.

Check out the events section tab on the main page for more clear information about server events that where detected.

Available Endpoints

Server Control

  • GET
    /start
    Start the game server
  • GET
    /stop
    Stop the game server

Information

  • GET
    /console
    Retrieve server console output
  • GET
    /events
    Retrieve Detected Events

Configuration

  • GET
    /config
    View server configuration
  • POST
    /saveconfig
    Save server configuration (Form data)

Save Management

  • GET
    /restore?index=123
    Restore from specified save index
  • GET
    /backups
    List available save files
